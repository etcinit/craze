haxy-1.0.1: A simple HTTP proxy server library

haxy-1.0.1: A simple HTTP proxy server library

A library for writing HTTP proxy servers with the focus on simplicity, flexibility and modularity. Allows arbitrary transformations on requests and responses and custom caching methods. It's up to the user to make sure that the message transformations are consistent with the HTTP specification. Note that the package versions follow Semantic Versioning model (semver.org).
